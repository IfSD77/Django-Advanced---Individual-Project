About this Portfolio

This is a personal collection of structural engineering projects I designed during my 8 years working in the UK.

The application was developed as part of the SoftUni Django Advanced Regular Exam – April 2026 and demonstrates advanced Django concepts: custom user model, group-based permissions, REST API, asynchronous tasks with Celery, and clean architectural design.

Technologies Used

  • Django 5.2.10
  • PostgreSQL (local development)
  • SQLite (production)
  • Django REST Framework
  • Celery + Redis (asynchronous tasks)
  • Bootstrap 5
  • Gunicorn + Nginx (deployment)
