More About This Portfolio

This project was built from scratch as part of the SoftUni Django Advanced course. It demonstrates advanced Django concepts including custom user models, group-based permissions, asynchronous tasks with Celery, and clean deployment on AWS EC2.

Key Features Implemented:

  • Full CRUD for Projects with Many-to-Many team participation
  • Role-based access (Administrators vs Viewers)
  • Graceful Celery email handling
  • Responsive Bootstrap 5 design
  • Custom 404 and 500 error pages
